# Toronto Renovation Pros Landing Page

This project is a simple landing page for a Toronto-based renovation company built with [React](https://react.dev/) and [Vite](https://vite.dev/).
The page includes hero, about, services, gallery, and contact sections inspired by [Senso Design's Toronto home renovation page](https://sensodesign.ca/toronto-home-renovations/).

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
